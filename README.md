# Expense Tracker
## Overview
Expense Tracker is a full-stack web application that allows users to
record, manage, and analyze their daily income and expenses.
The project follows a clean separation of frontend and backend.
## Features
- Add, edit, and delete income and expense records
- Categorize expenses for better tracking
- Filter expenses by date and category
- Visualize spending patterns using charts
- Separate frontend and backend architecture
## Tech Stack
### Frontend
- React.js
- JavaScript (ES6)
- HTML5
- CSS3
- Chart.js
### Backend
- Node.js
- Express.js
- REST APIs
- JSON-based data storage

## How to Run Locally
### Frontend
npm install
npm start
Runs on: http://localhost:3000
### Backend
npm install
node server.js
Runs on: http://localhost:5000
