# Expense Tracker – Backend

## Overview
This is the backend of the Expense Tracker application.
It provides RESTful APIs to manage income and expense data
and handles business logic for the application.
## Features
- REST APIs for expense management
- Create, read, update, and delete expense records
- Modular route and model structure
- JSON-based data storage
## Tech Stack
- Node.js
- Express.js
- JavaScript
## Folder Structure
backend/
├── data/
├── models/
├── routes/
├── server.js
├── package.json
├── package-lock.json
└── README.md
## How to Run
1. Install dependencies:
   npm install

2. Start the server:
   node server.js
## Notes
- `node_modules` is not included in the repository
- Environment variables and sensitive data are excluded
- Backend runs on http://localhost:5000
