# Expense Tracker – Frontend
## Overview
This is the frontend of the Expense Tracker application.
It provides a user-friendly interface to manage income and expenses
and visualize spending patterns.
## Features
- Add, edit, and delete expenses
- Track expenses by category
- Filter expenses by date
- View expense summaries using charts
- Responsive user interface
## Tech Stack
- React.js
- JavaScript (ES6)
- HTML5
- CSS3
- Chart.js
## Folder Structure
frontend/
├── public/
├── src/
├── package.json
├── package-lock.json
└── README.md
## How to Run
1. Install dependencies:
   npm install

2. Start the development server:
   npm start
## Notes
- `node_modules` is not included in the repository
- Run `npm install` before starting the application
- Frontend runs on http://localhost:3000
